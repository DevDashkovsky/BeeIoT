### 1. Улучшенный вывод информации в sensors.py
Все датчики теперь выводят **подробную информацию** при чтении:

#### DS18B20 (Температура):
- Адреса всех найденных устройств на шине
- Сырое значение температуры и округленное
- Статус (холодно/нормально/жарко)
- Поддержка нескольких датчиков на одной шине

#### KY-038 (Шум):
- Сырое значение ADC (0-4095)
- Статистика измерений (среднее, мин, макс)
- Уровень шума в дБ
- Статус (тихо/нормально/шумно)
- Цифровой выход (превышен порог или нет)

#### Датчик батареи:
- Напряжение в вольтах
- Уровень заряда в процентах
- Визуальный индикатор заряда
- Статус батареи

### Вариант 2: Тестирование в основной программе

Основная программа `main.py` уже имеет подробный вывод, но **требует:**
- ✗ Подключение SIM800L модуля
- ✗ GPRS соединение

**Не рекомендуется для первичного тестирования!**

## 🔧 Настройка для тестирования

### 1. Проверьте config.py

Убедитесь, что пины настроены правильно:

```python
# config.py
DS18B20_PIN = 4          # Пин для датчика температуры
KY038_ANALOG_PIN = 6     # Пин для датчика шума (ADC)
KY038_DIGITAL_PIN = 7    # Цифровой выход датчика шума
BATTERY_ADC_PIN = None   # Установите, если используете
```

### 2. Подключите датчики

#### DS18B20 (температура):
```
DS18B20      ESP32-S3
--------     --------
VCC     -->  3.3V
GND     -->  GND
DATA    -->  GPIO4 (+ резистор 4.7кОм к 3.3V)
```

#### KY-038 (шум):
```
KY-038       ESP32-S3
--------     --------
VCC     -->  3.3V
GND     -->  GND
A0      -->  GPIO6 (ADC)
D0      -->  GPIO7 (опционально)
```

### 3. Загрузите файлы на ESP32

Используйте `ampy`, `rshell` или Thonny IDE:

```bash
# Через ampy
ampy --port COM3 put config.py
ampy --port COM3 put sensors.py
ampy --port COM3 put test_sensors.py

# Запуск
ampy --port COM3 run test_sensors.py
```

## ⚠️ Важные замечания

### 1. Калибровка KY-038
Датчик шума требует калибровки! Текущие параметры примерные:
```python
NOISE_MIN_DB = 40.0   # Минимум
NOISE_MAX_DB = 120.0  # Максимум
```

**Как калибровать:**
1. Создайте тишину → запишите ADC значение → это ваш MIN
2. Создайте шум → запишите ADC значение → это ваш MAX
3. Измените параметры в config.py

### 2. Делитель напряжения для батареи
Если измеряете напряжение > 3.3V, нужен делитель:
```
Battery+ -->  R1  --> GPIO_PIN --> R2 --> GND
```

Коэффициент делителя: `K = (R1 + R2) / R2`

Измените код в `sensors.py`:
```python
voltage = (raw / self.adc_max) * self.vref * K  # Добавьте коэффициент
```

### 3. Несколько DS18B20
Можно подключить несколько датчиков на один пин!

Используйте:
```python
# Чтение всех датчиков
temps = temp_sensor.read_all_temperatures()
```

## 📞 Отладка проблем

### Датчик не найден:
- Проверьте подключение (VCC, GND, DATA)
- Проверьте номер пина в config.py
- Для DS18B20 обязателен подтягивающий резистор 4.7кОм

### Неправильные значения:
- Проверьте питание (3.3V стабильное)
- Проверьте калибровку датчика
- Увеличьте количество samples для усреднения

### Ошибки ADC:
- Убедитесь, что используете правильный ADC пин
- На ESP32-S3 используйте ADC1 (GPIO1-10)
- Не используйте ADC2 пины (конфликт с WiFi)
