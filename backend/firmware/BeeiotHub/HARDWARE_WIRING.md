# 🔌 Схема подключения BeeIoT - ESP32-S3 DevKitC-1

## 📋 Список компонентов

| Компонент | Модель | Количество | Назначение |
|-----------|--------|------------|------------|
| Микроконтроллер | ESP32-S3-DevKitC-1 | 1 шт | Основной контроллер |
| GSM/GPRS модуль | SIM800L | 1 шт | Связь с сервером |
| Датчик температуры | DS18B20 | 1 шт | Измерение температуры |
| Датчик звука | KY-038 | 1 шт | Измерение уровня шума |
| Батарея | Li-Ion 18650 3.7V 3400mAh | 1 шт | Питание |
| Резистор | 4.7 кОм | 1 шт | Pull-up для DS18B20 |
| Конденсатор | 470 мкФ 16V | 1 шт | Стабилизация питания SIM800L |

---

## 🔋 Схема питания

```
Li-Ion 18650 (3.7V)
    │
    ├──→ ESP32-S3 (5V pin)
    │
    └──→ SIM800L (VCC)
```

---

## 📐 Схема подключения

### 1. DS18B20 (датчик температуры) → ESP32

**DS18B20 распиновка:**
```
   ┌─────────┐
   │  ___    │
   │ /   \   │
   │ \___/   │
   │         │
   │ │ │ │   │
   └─┬─┬─┬───┘
     │ │ │
     1 2 3
     
1 - GND
2 - DATA
3 - VCC
```

**Подключение:**
| DS18B20 | ESP32-S3 |
|---------|----------|
| Pin 1 (GND) | GND |
| Pin 2 (DATA) | GPIO4 |
| Pin 3 (VCC) | 3V3 |

**Pull-up резистор 4.7 кОм между VCC (3V3) и DATA (GPIO4)**

---

### 2. KY-038 (датчик звука) → ESP32

**Подключение:**
| KY-038 | ESP32-S3 |
|--------|----------|
| GND | GND |
| VCC | 3V3 |
| DO | GPIO7 |
| AO | GPIO6 |

---

### 3. SIM800L (GSM/GPRS модуль) → ESP32

**Подключение:**
| SIM800L | ESP32-S3 / Питание |
|---------|-------------------|
| VCC | Li-Ion 18650 (+) |
| GND | GND + Li-Ion 18650 (-) |
| TXD | GPIO18 |
| RXD | GPIO17 |

**Конденсатор 470 мкФ между VCC и GND SIM800L (максимально близко к модулю)**

---

### 4. Батарея Li-Ion 18650

**Подключение:**
```
Li-Ion 18650 (3.7V 3400mAh)
    │
    ├──(+)──→ ESP32-S3 5V pin
    │
    ├──(+)──→ SIM800L VCC (с конденсатором 470µF)
    │
    └──(-)──→ Общая земля (GND)
```

## 📋 Пошаговая инструкция сборки

### Шаг 1: Питание
1. Припаять провода к батарее 18650 (красный к +, черный к -)
2. Подключить батарею + к ESP32 5V pin
3. Подключить батарею + к SIM800L VCC
4. Припаять конденсатор 470µF между VCC и GND SIM800L
5. Подключить батарею - к общей земле (ESP32 GND)

### Шаг 2: DS18B20
6. Подключить GND DS18B20 к ESP32 GND
7. Подключить VCC DS18B20 к ESP32 3V3
8. Подключить DATA DS18B20 к ESP32 GPIO4
9. Припаять резистор 4.7кОм между 3V3 и GPIO4

### Шаг 3: KY-038
10. Подключить GND KY-038 к ESP32 GND
11. Подключить VCC KY-038 к ESP32 3V3
12. Подключить AO к ESP32 GPIO6
13. Подключить DO к ESP32 GPIO7

### Шаг 4: SIM800L
14. Подключить GND SIM800L к общей земле
15. Подключить VCC SIM800L к батарее 18650 (+)
16. Подключить TX SIM800L к ESP32 GPIO18
17. Подключить RX SIM800L к ESP32 GPIO17
18. Вставить SIM-карту

### Шаг 5: Проверка
19. Проверить все соединения
20. Убедиться в отсутствии коротких замыканий
21. Подключить USB-C к ESP32 для прошивки

## 🧪 Тестирование

### Проверка датчиков в REPL
```python
# 1. DS18B20
from machine import Pin
import onewire, ds18x20
ds = ds18x20.DS18X20(onewire.OneWire(Pin(4)))
roms = ds.scan()
print("DS18B20 found:", len(roms))
ds.convert_temp()
import time; time.sleep_ms(750)
print("Temperature:", ds.read_temp(roms[0]))

# 2. KY-038
from machine import ADC, Pin
adc = ADC(Pin(6))
adc.atten(ADC.ATTN_11DB)
print("Noise level:", adc.read())

# 3. SIM800L
from machine import UART
uart = UART(1, baudrate=9600, tx=17, rx=18)
uart.write('AT\r\n')
time.sleep(1)
print("SIM800L:", uart.read())
```
