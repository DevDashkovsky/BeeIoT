# Инструкция по настройке SIM-карты для SIM800L

## Проблема: "SIM card not ready"

Эта ошибка означает, что модуль SIM800L не может работать с SIM-картой. Причины могут быть разные.

---

## 🔧 Решения по порядку

### 1. **Отключите PIN-код на SIM-карте** (САМАЯ ЧАСТАЯ ПРОБЛЕМА)

SIM800L не может работать с SIM-картами, защищенными PIN-кодом.

**Как отключить PIN:**

1. Вставьте SIM-карту в обычный телефон
2. Зайдите в **Настройки → Безопасность → SIM-карта**
3. Найдите опцию **"Блокировка SIM-карты"** или **"Запрос PIN-кода"**
4. **ОТКЛЮЧИТЕ** эту опцию (потребуется ввести текущий PIN)
5. Извлеките SIM-карту из телефона
6. Вставьте в модуль SIM800L

---

### 2. **Проверьте правильность установки SIM-карты**

- ✅ SIM-карта должна быть вставлена **до** включения питания
- ✅ Убедитесь, что SIM-карта правильно ориентирована (скошенный угол)
- ✅ Держатель SIM-карты должен быть **плотно закрыт/защелкнут**
- ✅ После установки SIM-карты сделайте **полный перезапуск** (выключите питание на 10 секунд)

**Формат SIM-карты:**
- SIM800L использует **mini-SIM** или **micro-SIM** (зависит от модуля)
- Если у вас nano-SIM, используйте адаптер

---

### 3. **Проверьте активацию SIM-карты**

SIM-карта должна быть:
- ✅ **Активирована** оператором
- ✅ На балансе должны быть **средства** (для передачи данных)
- ✅ Подключен **тариф с мобильным интернетом/GPRS**
- ✅ **Не заблокирована** оператором

**Как проверить:**
1. Вставьте SIM-карту в телефон
2. Проверьте, что телефон регистрируется в сети
3. Проверьте баланс: наберите команду проверки баланса оператора (например, `*100#` для МТС)
4. Попробуйте открыть интернет в телефоне

---

### 4. **Проверьте питание SIM800L**

SIM800L требует **стабильное питание 3.7-4.2V с током до 2A** во время работы.

**Типичные проблемы с питанием:**
- ❌ USB-порт компьютера (обычно дает максимум 500mA) - **НЕ ПОДХОДИТ**
- ❌ Тонкие провода питания
- ❌ Нет конденсаторов на линии питания

**Правильное питание:**
- ✅ Используйте **отдельный блок питания 4V/2A**
- ✅ Или Li-Ion аккумулятор 3.7V (18650)
- ✅ Подключите конденсатор **100-470 мкФ** параллельно питанию SIM800L
- ✅ Используйте короткие толстые провода (минимум 0.5мм²)

---

### 5. **Дайте модулю время на инициализацию**

После включения питания:
1. Подождите **10-15 секунд** перед первой командой
2. SIM800L должен моргнуть светодиодом несколько раз
3. Частота мигания LED показывает статус:
   - **1 раз в 1 сек** = не зарегистрирован в сети
   - **1 раз в 3 сек** = зарегистрирован в сети

---

### 6. **Проверьте антенну GSM**

- ✅ Антенна должна быть **подключена** к разъему SIM800L
- ✅ Проверьте качество соединения антенны
- ✅ Попробуйте в **другом месте** (ближе к окну, выше)

**Совет:** Для первого теста можно попробовать в месте с **хорошим покрытием сети** (рядом с окном или на улице).

---

### 7. **Проверьте формат SIM-карты**

- Некоторые **новые SIM-карты** могут не работать со старыми модулями
- Попробуйте **другую SIM-карту** (от другого оператора или старую)
- SIM-карты **только для LTE/4G** могут не работать с 2G модулями

---

## 📋 Тестирование SIM-карты в телефоне

Перед использованием в SIM800L, **обязательно** проверьте SIM-карту в телефоне:

1. ✅ Телефон регистрируется в сети оператора
2. ✅ Можно совершить звонок
3. ✅ Работает мобильный интернет (проверьте в браузере)
4. ✅ PIN-код отключен
5. ✅ Баланс положительный

Если хотя бы одно НЕ работает - **сначала исправьте это**.

## 🔍 Диагностика через улучшенный код

Я добавил улучшенную диагностику в код. Теперь при запуске вы увидите детальную информацию:

```
[SIM800L] SIM status response: +CPIN: READY
[SIM800L] [OK] SIM card is READY
[SIM800L] Checking operator registration...
[SIM800L] Operator: +COPS: 0,0,"MTS RUS",2
[SIM800L] Waiting for network registration (up to 60 sec)...
```

**Возможные сообщения об ошибках:**

### Ошибка: "SIM card requires PIN code"
```
[SIM800L] [ERROR] SIM card requires PIN code
[SIM800L] SOLUTION: Disable PIN on SIM card using phone
```
**Решение:** Отключите PIN-код (см. пункт 1)

### Ошибка: "SIM card not detected"
```
[SIM800L] [ERROR] SIM card not detected
[SIM800L] SOLUTION: Check if SIM card is properly inserted
```
**Решение:** Проверьте установку SIM-карты (см. пункт 2)

### Ошибка: "Network registration denied"
```
[SIM800L] [ERROR] Network registration denied
[SIM800L] SOLUTION: Check if SIM card is activated
```
**Решение:** SIM-карта не активирована или заблокирована оператором (см. пункт 3)

### Ошибка: "Network registration timeout"
```
[SIM800L] [ERROR] Network registration timeout (60 seconds)
```
**Решение:** 
- Проверьте антенну GSM
- Переместитесь в место с лучшим покрытием
- Проверьте питание модуля

## ⚙️ Настройка APN оператора

В файле `config.py` укажите правильный APN вашего оператора:

### Российские операторы:

```python
# МТС
APN = "internet.mts.ru"
APN_USER = "mts"
APN_PASSWORD = "mts"

# Билайн
APN = "internet.beeline.ru"
APN_USER = "beeline"
APN_PASSWORD = "beeline"

# Мегафон
APN = "internet"
APN_USER = ""
APN_PASSWORD = ""

# Теле2
APN = "internet.tele2.ru"
APN_USER = ""
APN_PASSWORD = ""

# Yota
APN = "internet.yota"
APN_USER = ""
APN_PASSWORD = ""
```

## 🧪 Режим тестирования (без SIM800L)

Если вы хотите сначала протестировать **только датчики** без модуля связи:

В файле `config.py` установите:
```python
TEST_MODE = True
SKIP_SIM800L = True
```

Программа будет работать в режиме тестирования - читать датчики каждые 5 секунд и выводить данные в консоль.

## 📞 Контрольный список

Перед запуском убедитесь:

- [ ] PIN-код на SIM-карте **ОТКЛЮЧЕН**
- [ ] SIM-карта **правильно вставлена** и держатель закрыт
- [ ] SIM-карта **активирована** и есть баланс
- [ ] SIM-карта **работает** в телефоне (звонки + интернет)
- [ ] SIM800L получает **достаточное питание** (4V/2A)
- [ ] Антенна **подключена** к SIM800L
- [ ] В `config.py` указан **правильный APN** оператора
- [ ] Подключены конденсаторы на линии питания SIM800L
- [ ] Испытание проходит в месте с **хорошим GSM сигналом**
